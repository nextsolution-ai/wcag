Accessibility Widget Demo

About

This is a demonstration of our accessibility widget. The widget provides various accessibility features to make websites more accessible to users with different needs.

Features

  • Easy to deploy
  • Keyboard shortcut support (CTRL + U)
  • Modern and clean design
  • Customizable styles
  • Responsive layout
  • WCAG compliant

Installation

Add the following script to your HTML file before the closing </body> tag:

<script src="https://nextsolution-ai.github.io/wcag/widget-deploy.min.js"></script>
